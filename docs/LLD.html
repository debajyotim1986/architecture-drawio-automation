chainCopilot▸unavailable_services.md▸match (alias-aware)▸rank by scenario▸cascade-skip restricted▸STOP · ask · wait
policy/unavailable_services.mdmd→disallowed list + alternatives the LLM reasons over

Cross-Cutting Invariantshold across every stage
Round-trip fidelity
parser.py preserves what SHAPE_STYLES can't rebuild: GCP image styles + swimlane colors (raw_style), edge entry/exit fractions, <Array> waypoints, and step badges — so load → mutate → save never loses prior work.
Path sandbox
DiagramStore / StoryStore resolve every name against their root and raise on escape — Copilot can't read or write outside diagrams/ & jira-stories/.
Idempotency
The 4 add_edge passes clear stale keys before recomputing; add_connector_table drops ctbl_ cells before re-laying — re-running converges, never duplicates.
Two hard gates
verify_mcp.py (exit 2 blocks delivery) and the add_connector_table zero-TBD pre-flight — a broken or half-documented diagram cannot ship.
Transport hygiene
stdout is reserved for MCP JSON-RPC; all logs/banners go to stderr. Handler exceptions are caught and returned as tool error text, never crashing the session.
Determinism & no-collide
Layout is computed from the constants above; filenames carry a UTC timestamp so re-running a prompt never overwrites a prior diagram.
